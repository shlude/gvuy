获取鼠标坐标

不能鼠标点击获取，鼠标移到相应位置再运行脚本，很鸡肋。

import pyautogui
print(pyautogui.position())

列表元素 → 复制到剪贴板 → 定位点击 → Ctrl+V 粘贴

依赖安装：

pip install pyautogui pyperclip
import pyautogui
import pyperclip
import time

# 你的字符串列表
data_list = [
    "第一条内容",
    "第二条内容",
    "第三条内容",
]

# 👉 目标位置（你需要改成你的输入框坐标）
x, y = 1000, 500

# 给你时间切换到 scrcpy 窗口
print("3秒后开始...")
time.sleep(3)

for text in data_list:
    pyperclip.copy(text)     # 复制到剪贴板
    pyautogui.click(x, y)    # 点击输入框
    pyautogui.hotkey('ctrl', 'v')  # 粘贴
    pyautogui.press('enter') # 回车（如需提交）
    time.sleep(1)            # 防止太快

print("完成")
